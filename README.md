# 🚀 PARASITE-COMPILER

A modern web-based plagiarism analysis and code similarity visualization platform built with **React, TypeScript, Vite, and Tailwind CSS**. The application provides an intuitive interface for uploading source code, analyzing similarities, and generating detailed visual reports.

---

## 📖 Overview

PARASITE-COMPILER is designed to simplify plagiarism detection by providing a fast, interactive, and user-friendly platform. It enables users to upload source code files, process them efficiently, visualize similarity metrics, and export reports for further analysis.

---

## ✨ Features

- 📂 Upload and analyze multiple source code files
- ⚡ Fast plagiarism detection and similarity analysis
- 📊 Interactive visualizations and result dashboards
- 📄 PDF report generation
- 🎨 Modern and responsive user interface
- 🔍 Syntax-aware processing
- 📱 Responsive design for desktop and mobile devices

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Shreya3501/PARASITE-COMPILER.git
```

Navigate into the project

```bash
cd PARASITE-COMPILER
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
